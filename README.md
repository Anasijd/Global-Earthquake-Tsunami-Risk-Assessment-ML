## Motivation
Natural disasters such as earthquakes and tsunamis have devastating impacts on human lives, infrastructure, and economies world wide. Predicting and assessing the risks associated with these events can significantly enhance preparedness and mitigaion efforts.

This project aims to leverage machine learning techniques to analyze global earthquake and tsunami data in order to:
- Identify high-risk regions across the globe.
- Develop predictive models for early warning systems.
- Support emergency apencies in disaster planning.
